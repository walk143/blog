点儿词库转Rime

记录将点儿词库转换为Rime格式的方法

下载点儿词库

从群313225526的群文件：点儿词库最新版本&历史版本 中下载最新版本的词库。最新为“点儿词库2021冬”

或从网站 gaokuan.ys168.com 下载

点儿词库的格式为编码 词条1 词条2……，分隔符为空格。同文所需格式为词条\t编码\t词频

  1. ~:生僻字词
  2. ^:用户词组
  3. !联想词组

command! AddFrequency call s:add_frequency_to_dict()
function! s:add_frequency_to_dict(line_sep=" ", val_sep='\t')
  function! s:load_line() closure
    " 加入\V 为了和 put 中统一，
    " 关于转义 :help \V
    let [k,v] = split(getline('.'), '\V'..a:line_sep)
    if has_key(numCount,k)
      " echo(k..len(numCount[k]))
      let numCount[k][v] = 1000 - 4 * len(numCount[k])
    else
      " echo(k..'---'..v)
      let numCount[k] = {}
      let numCount[k][v] = 1000
    endif
  endfunction

  " 存储编码，编码里面存储词组
  let numCount = {}
  execute 'g/\V'..a:line_sep..'/call s:load_line()'

  %d
  for [k,v] in items(numCount)
    for [value,frequency] in items(v)
      put = k..a:val_sep..value..a:val_sep..frequency
    endfor
  endfor
  1d

处理生僻字

点儿词库中以~开头的为生僻字，统一提取到单独的文件，方便统一调频、单独挂载之类的

1
2
3

# vim 中执行
# 移动~开头的到尾部，复制到新文件或调整编码
:g/^\~.*\n/normal ddGp

# vim 中执行
## 替换 $20 为空格
:%s/\$20/ /g
## 删除其余带$的，如命令直通车之类的
# 统计带$的数量
:%s/^.*\$.*\t.*$//gn
# 删除剩余的$
:%s/^.*\$.*\t.*$\n//g

至此处理结束。可将~开头的词条，移动到单独的一个文件。然后删除开头的~符
