Designed to load test functional behavior and measure performance

基础安装

  1. 官网
  2. 下载地址 
    1
2

    wget https://dlcdn.apache.org//jmeter/binaries/apache-jmeter-5.4.2.tgz
tar zxf apache-jmeter-5.4.2.tgz
  3. 运行 
    1

    ./bin/jmeter

使用

新建

  1. 新建一个配置，保存
  2. Test Plan>Add>Threads(Users)>Thread Group
    1. 线程数：代表用户数
    2. Ramp-up period：请求间隔时间
    3. Loop Count：循环多少次
  3. Thread Group>Add>Sampler>HTTP Request
  4. Test Plan>Add>Listner>Aggregate Report
    1. Average:平均响应时间，单位ms
