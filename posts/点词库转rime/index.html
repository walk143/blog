点儿词库转Rime

记录将点儿词库转换为Rime格式的方法

下载点儿词库

从群313225526的群文件：点儿词库最新版本&历史版本 中下载最新版本的词库。最新为“点儿词库2021冬”

或从网站 gaokuan.ys168.com 下载

点儿词库的格式为编码 词条1 词条2……，分隔符为空格。同文所需格式为词条\t编码\t词频

  1. ~:生僻字词
  2. ^:用户词组
  3. !联想词组

开始转换

先将点儿词库中的词条分开为单行。如编码 词条格式。

# 在vim中执行
# 将替换记录到宏中，有的编码，会有十几个词条，所以统一执行100次
# 一次替换一行
qr
:%s/^\(\S*\)\( \S*\)\( .*\).*$/\1\2\r\1\3
q
# 记录宏结束
100@r

将编码顺序修改为Rime格式

# vim 中执行
:%s/^\(\S*\) \(\S*\)$/\2\t\1\t4

处理生僻字

点儿词库中以~开头的为生僻字，统一提取到单独的文件，方便统一调频、单独挂载之类的

# vim 中执行
# 移动~开头的到尾部，复制到新文件或调整编码
:g/^\~.*\n/normal ddGp

$符号单独处理

# vim 中执行
## 替换 $20 为空格
:%s/\$20/ /g
## 删除其余带$的，如命令直通车之类的
# 统计带$的数量
:%s/^.*\$.*\t.*$//gn
# 删除剩余的$
:%s/^.*\$.*\t.*$\n//g

至此处理结束。可将~开头的词条，移动到单独的一个文件。然后删除开头的~符

# vim 中执行
# 删除生僻字的开头 ~ ，并将词频从4改成3
:%s/^\~\(.*\)\t4/\1\t3/g
