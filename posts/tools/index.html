moco

https://github.com/dreamhead/moco

用于mock一些rest请求

编译

git clone git@github.com:dreamhead/moco.git
./gradlew build

#Build uberjar
./gradlew uberjar

# Check code before commit
./gradlew check

生成后目录在./moco-runner/build/libs/moco-runner-1.2.0.jar
